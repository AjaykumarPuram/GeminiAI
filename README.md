# Gemini AI - Java Demo

This is one of my first Java projects where I got to play around 
with AI. It connects to Google's Gemini AI and lets you send a 
message and get a response back — all from your own Java code.

## How it works

You write a message in the code, run it, and Gemini AI replies 
back in your console. Pretty cool for a first project!

## Built with

- Java
- Maven
- OkHttp — to send the message over the internet
- Google Gemini API — the AI brain behind it all

## How to run it yourself

1. Clone this repo
2. Head over to [aistudio.google.com](https://aistudio.google.com) 
   and grab a free API key
3. Add your key as an environment variable called `GEMINI_API_KEY`
4. Open it in IntelliJ and hit run!
